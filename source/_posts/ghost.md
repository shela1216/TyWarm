《天諭手遊》中每位玩家最多可以裝備四種神格，不同位置的屬性加成按不同百分比依次遞減。那麼不同職業應該怎樣選擇神格呢？又有哪些不同的神格呢？一起來看看吧！ <!-- more -->

## 總體分析

在游戲中，每個神格都有自己的裝備以及收集兩種屬性。

每位玩家最多可以裝備四種神格，不同位置的屬性加成按不同百分比依次遞減，且只有前兩格裝備神格的被動技能才會生效。

第一格神格屬性值100%生效，第二格生效70%的數值，第三格生效50%，第四格生效30%。

其他未裝備神格在收集後雖然本身的裝備屬性值不會生效，但是由於這些神格都存在收集屬性，且這方面數值會生效，因此玩家戰力也會有所增長。

## PVE神格選擇

![PVE神格搭配](https://www.steamxo.com/wp-content/uploads/2021/03/FmR27w130718_1834106.jpg)

**PVE神格解析**
1. 藍色神格中只有一個日之晶是完全PVE的選擇，前期沒有其他品質的神格時，可以先拿著用。
2. 紫色神格中勇之晶偏向防禦類，心之晶偏向攻擊類，不過，由於心之晶容易獲取（綁玉購買），所以優先推薦心之晶以及日之晶的組合。
3. 金色神格中，玄水，金剛全體適用，副本中的非拉怪職業優先考慮梅心，拉怪職業可以佩戴和音，輸出以及治療職業可以先考慮帝翼。

## PVP神格選擇

在PVP中，由於不同職業的屬性以及戰鬥方式等不同，我們在神格的選擇方面也會有所不同，而且，即使是像在神耀競技這種平衡玩家屬性的玩法中，神格也是會發揮作用的。

不同職業適配神格

### 聖堂推薦神格及分析

+ 雙金

    推薦**帝翼+金剛** : 利用第一個槽位100%帝翼被動加成，按生命值增強傷害率和治療力及傷害減免，第二個槽位金剛累積循環被動70%加成

    **玄水+帝翼** : 玄水可以加快技能冷卻時間，這裡按照冒險家自己用技能習慣選擇鑲嵌第幾槽位
+ 金紫

    則可以選擇**金剛+生之晶** : 金剛用於累積循環與生之晶配合消耗對方

+ 雙紫

    推薦**心之晶+生之晶** : 心之金可以所有傷害無視目標防禦力再配合生之晶移動1米恢復生命值消耗對方為目的出擊

### 玉虛推薦神格及分析

+ 雙金推薦

    **帝翼+梅心** : 利用第一個槽位100%帝翼被動加成，技巧型玉虛利用梅心從背後命中目標提升傷害

    **帝翼+玄水** : 利用帝翼提高傷害及治療深度，生命值低時再獲得少量傷害減免，技能型玉虛則可以選擇配合玄水加快技能冷卻

    **帝翼+金剛** : 兩面加強傷害，發揮玉虛高額傷害

+ 金紫

    可以選擇**玄鏡+心之晶** : 若3秒無傷利用玄鏡增加傷害及治療強度，心之晶無視對方防禦力，這裡推薦技巧型玉虛使用

+ 雙紫

    推薦**心之晶+生之晶** : 心之金可以所有傷害無視目標防禦力再配合生之晶移動1米恢復生命值消耗對方為目的出擊

### 光刃推薦神格及分析

+ 雙金推薦

    **帝翼+梅心** : 利用第一個槽位100%帝翼被動加成，再利用梅心從背後命中目標提升傷害
+ 金紫

    可以選擇**帝翼+生之晶** : 提高傷害及傷害減免，同時通過位移恢復生命值

+ 雙紫

    推薦**心之晶+生之晶** : 心之金可以所有傷害無視目標防禦力再配合生之晶移動1米恢復生命值消耗對方為目的出擊

### 靈瓏推薦神格及分析

+ 雙金推薦

    **金剛+玄鏡** : 精鋼累積循環增強治療、傷害減免和增加移速及加快輕功恢復速度，配合玄鏡未受傷害3秒提升自身的傷害和治療加深

+ 金紫推薦

    **帝翼+魄之晶** : 增強傷害及治療強度還有少量傷害減免配合當自身生命值低於30%時沉默周圍5米內的所有目標，持續3秒，一分鐘內最多觸發一次

+ 雙紫推薦

    **心之晶+生之晶** : 心之金可以所有傷害無視目標防禦力再配合生之晶移動1米恢復生命值消耗對方為目的出擊]

    **生之晶+勇之晶** : 入戰後獲得一個盾，盾可持續一分鐘且一分鐘最多觸發一次配合位移恢復生命值使用

    則可以選擇**金剛+生之晶** : 金剛用於累積循環和生之晶配合使用消耗對方生命值

## 神格培養順序

**前期**白嫖黨推薦培養藍色神格<font color="red">日之晶</font>，紫色神格可以培養<font color="red">心之晶和勇之晶</font>，當然<font color="red">有能力培養生之晶就培養生之晶</font>

**主PVP**可以選擇<font color="red">生之晶和心之晶</font>（每移動1米恢復自身0.33%最大生命值但玩家對抗時會衰減且所有傷害無視目標3.0%防禦力）PVE則可以把<font color="red">生之晶換成韌之晶</font>（心之晶移動回血，韌之晶移動消耗層數）

**後期PVE**金色神格推薦：<font color="red">梅心、玄鏡、帝翼</font>（輸出位考慮玄境且致命加成很高），**PVP推薦**<font color="red">玄水、金剛、帝翼、和音</font>（玄水在團戰單挑中都十分亮眼，和音這裡表現不是很突出可考慮添加）

+ pvp聖堂雙紫神格推薦生之晶與勇之晶
+ 微氪聖堂可以考慮生之晶與淚之晶
+ 靈瓏雙紫神格推薦生之晶與勇之晶，禦魂靈瓏可以選擇心之晶
+ 天昭推薦金剛、玄境（個人看法這里天命也可以考慮）】

金色神格里金剛、和音、玄鏡、天命的屬性較弱，帝翼與梅心會比前四者強一些，再強就是玄水與君臨，同培養度紫色神格加成沒有比金色神格加成高，如果可以還是建議培養一個金色神格（尤為註意神格技能再選擇進行培養）

同等級的神格全部通過修煉可以提升收集屬性和收集屬性，同時增加的戰力會比升級前兩個神格提升的戰力更多

## 神格碎晶與神格之力提取

神格的進階有兩種方式，突破以及精魄修煉。

1. 第一種突破是只要在收集了神格的前提下，再收集更多的同神格碎片，並消耗一定量的雲券即可完成進階。
2. 第二種是通過神格碎晶直接修煉。目前已知碎晶的獲取方式有青麟鎮妖，怒鱗商店，園游會商店，精魄提取以及神格助力袋獲取。
   
>另外，去世界地圖上找尋神格進行提取，提取出的神格之力同樣可以修煉神格，不過在提取之前，記得勾選一下需要修煉的神格。
>神格每天可以提取10次，前4次可以獲得額外50%的神格之力。
